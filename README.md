<h1><center>openpaymentstui</h1></center>

### Current Features
- Connect wallets
- Send payments 
- View latest payments
- Flow engine

### Development

> [!WARNING]  
> Only tested on macOS

To install dependencies:

```bash
bun install
```

To run:

```bash
bun start
```
